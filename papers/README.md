# Accepted Papers PDFs

Place PDF files of accepted workshop papers here, then in `accepted_papers.yaml` use the **raw** GitHub URL so the link opens the PDF directly (not the GitHub file page):

```
https://raw.githubusercontent.com/SAI-Lab-NYU/VisArch/main/papers/你的文件名.pdf
```

Do **not** use `github.com/.../blob/main/...` — that opens GitHub’s page instead of the PDF.
